# ICTN6875-Capstone-David-Corley
ICTN 6875 Network Automation Capstone Project for David Corley

This project is a model-driven network automation lab using NETCONF and Python,
designed for undergraduate networking students.

The lab teaches:
- The difference between CLI-based and model-driven automation
- NETCONF-based configuration retrieval
- Safe interface-level configuration changes
- Post-change verification and validation

Author/Creator: David Corley
Course: ICTN 6875
